Help & Documentation
1. Getting Started

OneDrive Folder: Use this OneDrive folder Wrike AI Assistant and Click on Add shortcut to OneDrive link so that you can access them in your computer.

Install: Load via chrome://extensions → Developer mode → Load unpacked.

AI Provider / API Key

Open Settings (⚙) → AI Provider. Choose a provider (Claude, OpenAI, Gemini, or Ollama), enter your API key and select a model. Click Test Connection to verify.

Wrike Token

Open Settings (⚙) → Wrike Connection. Paste your Wrike Access Token and click Save Token. The badge turns green when connected.

To get a token: Wrike → Profile → Apps & Integrations → API → Create token.

Your Profile

Go to Settings → Your Profile. Enter your name, role, and standing instructions. These personalise every AI response.

2. Analyzing a Ticket

Navigate to any Wrike task URL. If Auto-analyze is enabled the extension starts automatically; otherwise click Analyze.

Context Panel

  • Extra context — paste emails, Teams messages, or meeting notes.
  • Attach files — images, PDFs, text, CSV (max 5 MB each, up to 10 files). You can also paste a screenshot directly.
  • Fetch URLs — paste a URL and click Fetch to pull its content into the analysis. Links found in the ticket are listed automatically. Times out after 12 s.
  • Tone — Conversational, Formal, or Assertive.

Click Re-analyze to regenerate all tabs with updated context. Use Refresh ↺ in the header to bypass the cache.

Update Banner

If a ticket has changed since your last analysis, a banner appears at the top. Click Re-analyze in the banner to refresh, or dismiss it.

Recent Wrike Comments

The most recent comments on the ticket are shown below the ticket title, so you have full context before generating a response.

3. Tabs & Responses
  • Summary — concise overview. Always shown.
  • Ask — free-form follow-up chat with full history per ticket.
  • Next Steps — prioritised actions. Editable with version history.
  • My Comment — draft a Wrike comment. Post directly with Post to Wrike.

Tabs generate on demand when clicked. Editable tabs (Next Steps, My Comment, Meeting Request, Meeting Script) support custom edit instructions and version rollback via the version bar.

Toolbar actions

  • Regen. All — regenerate every visible tab in one click.
  • Export .md — download all tab content as a Markdown file.
  • Copy All — copy all tab content to the clipboard.

Set persistent per-tab style instructions in Settings → Behavior & Display → Tab Response Styles. Add your own tabs in Settings → Custom Tabs. Hide unused tabs in Settings → Behavior & Display → Visible Tabs.

4. Recent Analyses & Comparison

Click 📋 in the header to open Recent Analyses. Every ticket you analyse is saved here automatically.

  • Click any entry to reload that full analysis instantly — no AI call needed.
  • Click Compare, select 2–5 analyses, then click Compare to see an AI comparison across all selected tickets.
  • Click Clear all to wipe the history and free storage.
5. My Wrike Tasks

Click 🎯 in the header to open the task browser. Shows tasks for you and your team members. Completed tasks are hidden by default — select a completed status from the Status dropdown to show them.

Filters & Reset

  • Search — type to filter tasks by title in real time.
  • Priority — filter by Wrike priority level.
  • Status, Folder, Assignee — multi-select dropdowns with Select all / Clear selection. Folder dropdown always shows your Saved Folders. Status dropdown is grouped by workflow.
  • Date — filter by last modified or completed within a time window.
  • ✕ Reset — resets all filters and assignee back to defaults in one click.
  • Pin folders in Settings → Wrike Connection → Saved Folders. Tasks are always scoped to saved folders regardless of which assignee is selected.
  • Add team members in Settings → Wrike Connection → Team Members.

Sort & Group

Sort by Last Modified, Priority, Status, Title, Channel, or Optimisation Type. Group by Channel, Solution Area, Folder, Status, or any extra custom fields you have configured in Settings → My Tasks Fields — click any group header to collapse or expand it.

Presets

Save the current filter combination as a named preset (up to 6). Click a preset chip to restore that filter state instantly. Click ✕ on a chip to delete it.

Starring tasks for Newsletter

Hover over any task card to reveal a ☆ star button in the top-right corner. Click it to pin that task for the newsletter — the card highlights and the star fills (★). The Newsletter button badge shows how many tasks are starred. Stars reset when you reload the task list.

Workload Summary

Click 📊 Summary to AI-analyse the currently filtered tasks. Task titles link directly to Wrike. After the summary loads, a follow-up chat appears — ask questions about the workload or specific tasks.

6. Newsletter

Click 📰 Newsletter in the My Tasks header to generate a Digital Engine Optimisation Newsletter from your tasks.

  • If no tasks are starred, the newsletter covers all currently filtered tasks.
  • If any tasks are starred (★), the newsletter covers only those starred tasks. The button badge shows the count: 📰 Newsletter ★3.

What the newsletter includes

  • Date and introduction paragraph.
  • One section per task — title linked to the Wrike ticket, status, folder, key updates, and people involved.
  • Special Thanks — automatically lists task followers and Watchers custom field contacts, deduplicated across all tasks.

Controls

  • ↻ Regenerate — re-run the newsletter generation from scratch.
  • Copy — copy the full newsletter to the clipboard.
  • A follow-up chat appears after generation — use it to adjust tone, add a section, or ask questions.

The newsletter fetches full task details and contact names from Wrike in 3 steps. Contact names are cached for the session so subsequent runs are faster.

7. Settings Reference
  • Your Profile — name, role, team context, standing instructions. Included in every AI prompt.
  • Wrike Connection — token or OAuth, space name, team members, saved folders.
  • AI Provider — choose provider (Claude / OpenAI / Gemini / Ollama), model, API key, cache duration (30 min / 1 hour / 4 hours / Forever).
  • Behavior & Display — auto-analyze, visible tabs, per-tab response styles.
  • Custom Tabs — add analysis tabs with a custom name and prompt.
  • My Tasks Fields — configure custom field names for grouping, sorting, and tags. Click ↺ Load from Wrike to pick fields from a searchable list instead of typing. Add extra fields beyond the fixed 3 (Channel, Solution Area, Opt. Type) — they appear as additional grouping options and task tags. An amber warning appears in the task list if a configured field name doesn't match your workspace.
  • Cache & Data — clear cached analyses, export/import all settings as JSON (includes custom fields, saved folders, team members, presets).
8. Keyboard Shortcuts
  • Cmd+Shift+W (Mac) / Ctrl+Shift+W (Windows) — open or close the extension.

Change shortcuts at chrome://extensions/shortcuts.

9. Troubleshooting
  • "Connect Wrike in Settings first" — token missing or expired. Re-paste a fresh token in Settings → Wrike Connection.
  • "Could not fetch tasks" — verify the token badge is green, then click Refresh inside the task panel.
  • "Could not identify your account" — use a personal token, not a service-account token.
  • URL fetch timeout — some sites block CORS or require login. Attach the content as a file instead.
  • Blank or failed analysis — go to Settings → AI Provider → Test Connection. Check the provider is configured correctly with a valid API key.
  • "Not a Wrike ticket" — the extension only activates on task-detail URLs, not dashboards or folders.
  • Seeing stale analysis — click Refresh ↺ in the header, or clear the cache in Settings → Cache & Data.
  • Newsletter Special Thanks missing people — ensure the Watchers custom field name in Settings → My Tasks Fields matches your workspace exactly.
  • Starred tasks disappeared — stars are session-only and reset when you reload the task list. Star tasks again before generating the newsletter.

If issues persist, reload the extension at chrome://extensions → Wrike AI Assistant → reload icon.
